#### QUESTIONS: KERNEL

>B7: At the entry of pintos_init(), what is the value of expression `init_page_dir[pd_no(ptov(0))]` in hexadecimal format?
Ans.
執行指令 p=print /x代表hexa 
(gdb) p /x init_page_dir[pd_no(ptov(0))]
    => 0xc000efcf:  int3
    => 0xc000efcf:  int3
    $1 = 0x0  答案0
解釋
init_page_dir是一個global variable 這時候還沒初始化，放在BSS段預設0 是page dir的初始目錄 VM會詳細

$1是print後的序號 每次p都會提供編號 後續可用編號方便計算

=>這兩行是 GDB 的行為標記
int3 -> 中斷3 網站第一種CPU-generate
這行是GDB設B時插入得 CPU運行看到這行會觸trap然後停止
